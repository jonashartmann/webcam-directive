Webcam directive

 Source on Github  Download v3.0.0 
bower install webcam-directive

Depends on angularjs

Directive based on the WebRTC API.
Which browsers are supported?

Add module "webcam" as dependency

angular.module('myapp', ['webcam']);

Then in the HTML

<webcam></webcam>

With callbacks

<webcam on-stream="onStream(stream)"
        on-access-denied="onError(err)"
        on-streaming="onSuccess()">
</webcam>

With placeholder

<webcam placeholder="'img/ajax-loader.gif'">
